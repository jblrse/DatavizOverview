    Grammar of Graphics

  • Concept by Leeland Wilkinson from 'The Grammar of Graphics' (2010)
  • Attempts standard syntax for defining graphics
  • Syntax varies based on language (e.g. R, Python, Javascript)
  • Breaks graph down into reusable components

    • Example:

  • Scatterplot => Axis, Scale, Point, Data
  • Point => Size, Shape, Opacity
